# macOS first boot fake wizard 
Based on [Electron Framework](http://electron.atom.io/).  
Tested on Ubuntu 17.04 and MacBookPro11,4 2015 Motherboard 820-00138-A

### Install and Run 

Install dependencies  
`npm install`

Run  
`npm start`
